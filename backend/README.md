# ForgeWrite

ForgeWrite is an AI-powered prompt compiler that transforms vague user input into structured, high-quality prompts optimized for large language models.

## Features

- Intent classification (Domain / Persona / Task)
- Structured JSON output
- Deterministic prompt compilation
- Gemini 2.5 Flash integration
- FastAPI backend

## Tech Stack

- FastAPI
- Gemini API (google-genai)
- Pydantic
- Python
